## 四、安全配置建议

### 当前安全状态

```
✅ Gateway绑定loopback（仅本地访问）
✅ 使用token认证
✅ Telegram使用pairing策略
⚠️ 有3个安全警告需要注意
```

### 建议的安全增强

1. **配置groupAllowFrom** - 当前groupPolicy是allowlist但列表为空
   ```json5
   {
     channels: {
       telegram: {
         groupAllowFrom: ["<your-group-id>"]
       }
     }
   }
   ```

2. **启用沙箱** - 如果要运行不可信代码
   ```json5
   {
     agents: {
       defaults: {
         sandbox: { mode: "non-main" }
       }
     }
   }
   ```

3. **限制工具** - 按需开放
   ```json5
   {
     tools: {
       profile: "coding",
       deny: ["browser"]  // 如果不需要
     }
   }
   ```

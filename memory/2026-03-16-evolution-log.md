## 13:15 - OpenCLI 安装成功 ✅

**已安装：** `@jackwener/opencli`

**功能：** 把任何网站变成命令行工具
- 支持 18+ 网站
- 复用 Chrome 登录状态
- 支持多种输出格式（json/yaml/md/csv）

**可用命令：**
- `opencli list` - 列出所有命令
- `opencli bilibili hot` - B站热榜
- `opencli zhihu hot` - 知乎热榜
- `opencli xiaohongshu search "关键词"` - 小红书搜索
- `opencli twitter search "关键词"` - Twitter搜索

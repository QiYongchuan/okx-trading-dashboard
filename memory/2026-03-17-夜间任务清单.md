### 实现方式

#### 方式1：通过 Agent 绑定（推荐）

在 `openclaw.json` 中配置多个 agent：

```json5
{
  agents: {
    list: [
      {
        id: "main",
        default: true,
        model: { primary: "zai/glm-5" }
      },
      {
        id: "coder",
        model: { primary: "openai-codex/gpt-5.2-codex" },
        workspace: "~/.openclaw/workspace-coder"
      },
      {
        id: "analyst",
        model: { primary: "anthropic/claude-opus-4-5" },
        workspace: "~/.openclaw/workspace-analyst"
      }
    ]
  },
  bindings: [
    {
      agentId: "main",
      match: { channel: "telegram" }
    },
    {
      agentId: "coder",
      match: { channel: "telegram", messageContains: ["写代码", "codex", "debug"] }
    }
  ]
}
```

#### 方式2：通过子代理（Subagent）

使用 `sessions_spawn` 工具调用 coding-agent：

```
Kevin: 用 codex 帮我写一个爬虫
我: → sessions_spawn(runtime: "acp", agentId: "codex", task: "写一个爬虫...")
```

## 一、内容抓取工具

### 1. X/Twitter 抓取

**方法1：r.jina.ai（推荐）**
```
https://r.jina.ai/https://x.com/用户名/status/推文ID
```
- ✅ 免费，20次/分钟
- ✅ 获取正文 + 图片链接
- ✅ 支持抓取评论

**方法2：feedgrab（批量）**
```bash
cd ~/tools/feedgrab
python3.11 -m feedgrab.cli https://x.com/user/status/123
```
- ✅ 支持批量抓取书签
- ✅ 支持抓取用户全部推文
- ⚠️ 需要配置 Cookie

**方法3：OpenCLI**
```bash
opencli twitter search "关键词"
opencli twitter bookmarks
```
- ✅ 复用 Chrome 登录状态
- ✅ 支持多种输出格式

---

### 2. 微信公众号抓取

**方法1：feedgrab（推荐）**
```bash
cd ~/tools/feedgrab
python3.11 -m feedgrab.cli "https://mp.weixin.qq.com/s/文章链接"
```
- ✅ 已安装
- ✅ 支持浏览器自动化
- ⚠️ 需要登录（`feedgrab login wechat`）

**方法2：搜索公众号文章**
```bash
python3.11 -m feedgrab.cli mpweixin-so "关键词"
```

**方法3：按账号批量抓取**
```bash
python3.11 -m feedgrab.cli mpweixin-id "公众号名称"
```
- ⚠️ 需要先登录

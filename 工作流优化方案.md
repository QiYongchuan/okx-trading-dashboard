## 工作流设计方案

### 方案 1：手动触发（简单）

```
用户在飞书/Telegram说："用 cc 帮我写一个 xxx"
    ↓
贾维斯接收指令
    ↓
贾维斯在本地执行：cc "写一个 xxx"
    ↓
cc 生成代码
    ↓
贾维斯把结果返回给用户
```

**优点**：
- 简单，不需要额外配置
- 用户完全控制

**缺点**：
- 需要手动触发
- 不是自动化的

---

### 方案 2：自动触发 + ACP（推荐）

```
用户在飞书/Telegram说："帮我实现 xxx 功能"
    ↓
贾维斯判断是否需要编码
    ↓
如果是，使用 sessions_spawn (runtime="acp", agentId="claude-code")
    ↓
Claude Code 在隔离环境执行
    ↓
完成后自动返回结果
```

**优点**：
- 完全自动化
- 隔离环境，安全
- 支持复杂任务

**缺点**：
- 需要配置 ACP

---

### 方案 3：混合方案（最灵活）

```
用户在飞书/Telegram说："帮我 xxx"
    ↓
贾维斯判断任务类型：
    - 简单任务 → 直接在本地执行 cc
    - 复杂任务 → 用 sessions_spawn (ACP)
    - 长时间任务 → 用 sessions_spawn (subagent)
    ↓
返回结果
```

## 技术细节

### Claude Code CLI 参数

```bash
# 基本使用
claude "写一个 Python 脚本"

# 指定目录
cd ~/myproject && claude "重构这个项目"

# 使用特定模型
claude --model claude-opus-4 "写代码"

# 输出到文件
claude "写代码" > output.txt
```

---

### OpenClaw ACP 配置

在 `openclaw.json` 中添加：

```json
{
  "acp": {
    "allowedAgents": ["claude-code", "codex", "gemini"],
    "defaultAgent": "claude-code"
  }
}
```

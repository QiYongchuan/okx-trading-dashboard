# Self-Improving Skill - 快速入门

让 AI 自动记住你的偏好，不再重复讲解。

## 🎯 这个 skill 做什么？

1. **自动识别**你表达的偏好和规则
2. **结构化记录**这些偏好
3. **主动应用**在未来任务中
4. **明确告知**正在遵循什么规则

## 🚀 如何使用

### 表达偏好

直接告诉 AI 你的规则：

```
"不要让我手动复制文档，永远不要"
"下次不要让我重复讲这个"
"记住，我更喜欢简洁的回复"
```

### AI 会做什么

1. 识别你在表达偏好
2. 记录到 `memory/preferences.json`
3. 在相关任务中自动应用
4. 在回复开头明确声明

### 示例对话

```
你: "帮我整理一下这个飞书文档"

AI: 🎯 我注意到你的偏好：
- 永远不要让你手动复制文档内容

我会直接通过 API 获取文档内容，不需要你手动复制。

[开始通过 feishu-fetch-doc skill 获取文档...]
```

## 💡 设计理念

### 为什么需要这个？

1. **AI 注意力收窄** - AI 执行任务时不会主动翻阅远处的文档
2. **"Memory" 的误导** - 存进去不等于记住了，需要主动应用
3. **重复的痛苦** - 你不应该反复讲解同一件事

### 核心原则

- **主动 > 被动** - 不等提醒，主动应用
- **透明 > 隐蔽** - 明确告知在遵循什么
- **可解释 > 黑盒** - 你能理解 AI 的行为

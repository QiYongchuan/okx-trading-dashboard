# Self-Improving Skill - 快速参考

## 🎯 核心功能

**让 AI 自动记住你的偏好，不再重复讲解。**

## 📝 表达偏好的方式

### 强触发（立即记录）
```
"永远不要..."
"下次不要..."
"记住，..."
"以后都这样..."
"不要让我重复讲..."
```

### 弱触发（询问后记录）
```
"我希望..."
"我倾向于..."
"能不能..."
"最好是..."
```

## 🗂️ 偏好分类

| 分类 | 说明 | 示例 |
|------|------|------|
| workflow | 工作流规则 | "不要手动复制文档" |
| communication | 沟通规则 | "群里回复要简短" |
| tools | 工具使用规则 | "优先使用 Obsidian" |

## ⚠️ 严重程度

| 级别 | 行为 |
|------|------|
| **critical** | 立即阻止违反操作 |
| **high** | 警告并询问 |
| **medium** | 记录但继续 |
| **low** | 提示但不阻止 |

## 🎯 应用流程

```
1. 任务开始
   ↓
2. 读取 preferences.json
   ↓
3. 检查任务关键词
   ↓
4. 匹配偏好？
   ├─ 是 → 声明偏好 → 应用规则 → 执行任务
   └─ 否 → 直接执行任务
```

## 💡 使用技巧

### ✅ 好的表达方式
```
"记住，我讨厌 X"           → 明确、强烈
"以后都这样..."           → 长期规则
"永远不要..."             → critical 级别
```

### ❌ 不好的表达方式
```
"这次能不能..."           → 一次性需求
"也许可以..."            → 不够明确
"我随便..."              → 没有偏好
```

## 🔧 管理偏好

### 查看所有偏好
```
"显示我的所有偏好"
```

### 更新偏好
```
"其实 X 也可以，只要..."
```

### 删除偏好
